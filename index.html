alfred-uri-parser

URI parsing workflow for Alfred

Usage:

  • Copy any URI (Example: https://github.com/dkulkarni/alfred-uri-parser/edit/master/README.md) to clipboard
  • On Alfred, type uri
  • Press ENTER to copy the segment of the URI
